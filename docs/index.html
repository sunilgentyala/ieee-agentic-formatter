IEEE Agentic Formatter

Transform raw research notes, loose drafts, or Markdown files into a strictly compliant, two-column IEEE Conference Paper (.docx) in seconds -- powered by Claude AI and python-docx.

GitHub Repository Quick Start

What it does

One pipeline from unstructured text to publication-ready IEEE paper. No manual formatting. No template wrestling.

🧠

Agentic Parsing

Claude (claude-opus-4-8) with adaptive thinking and forced tool use extracts title, authors, abstract, keywords, sections, and references from any raw input.

📄

Strict IEEE Layout

Letter-size, mirrored margins, single-column title block, continuous-break into two justified columns, Times New Roman typography, Roman-numeral headings.

📄

Multi-format Input

Paste raw text, upload a .txt, .md, or .docx file. The agent infers and fills any missing sections from context.

⬇️

One-click Download

Generates an in-memory .docx via python-docx and delivers it as a browser download -- no temp files, no cloud storage.

IEEE Formatting Specification

Every output is validated against these constraints before download.

PropertyValue
Page sizeLetter (8.5 x 11 in)
Margins1 in all sides, mirrored (odd/even pages)
Title fontTimes New Roman 24 pt, centered
Author blockTimes New Roman 10 pt (name) / 10 pt italic (affiliation)
Abstract / KeywordsTimes New Roman 9 pt bold-italic label + 9 pt italic body
Body columns2 equal columns, 0.32 in gap, continuous section break
Body fontTimes New Roman 10 pt, justified, 0.15 in first-line indent, 12 pt exact leading
Section headingsLevel 1: Roman numeral (I. INTRODUCTION) centered bold; Level 2: Letter (A. Background) left italic
ReferencesTimes New Roman 8 pt, hanging indent, IEEE bracket style

Quick Start

Up and running in under two minutes.

  1. Clone and install git clone https://github.com/sunilgentyala/ieee-agentic-formatter.git && pip install -r requirements.txt
  2. Add your API key Create a .env file: ANTHROPIC_API_KEY=sk-ant-...
  3. Launch the app streamlit run app.py
  4. Paste or upload your content and click Generate Download the IEEE-compliant .docx instantly. 

# Clone
git clone https://github.com/sunilgentyala/ieee-agentic-formatter.git
cd ieee-agentic-formatter

# Install
pip install -r requirements.txt

# Configure
echo "ANTHROPIC_API_KEY=sk-ant-..." > .env

# Run
streamlit run app.py

Architecture

Three clean modules, zero circular dependencies.

agent/

models.py -- Pydantic v2 data models (IEEEPaper, Author, Section).

text_parser.py -- Anthropic streaming API with tool_choice forced to structure_ieee_paper. Adaptive thinking enabled for complex inputs.

formatter/

docx_engine.py -- python-docx + raw OOXML (OxmlElement) to inject w:mirrorMargins, two-column w:cols, and exact-leading paragraph spacing without a Word template dependency.

Stack

Claude claude-opus-4-8

Anthropic's most capable model with adaptive thinking and structured tool-use output.

python-docx + OOXML

Fine-grained Word document control via both the high-level API and raw XML injection.

Pydantic v2

Schema-validated structured output ensures the formatter always receives well-formed data.

Streamlit

Minimal, fast UI for text input, file upload, and in-browser .docx download.
